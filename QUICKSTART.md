## Installation

```bash
# 1. Install dependencies
npm install

# 2. Copy environment template
cp .env.example .env

# 3. Edit .env and add:
# - WalletConnect Project ID (get from https://cloud.walletconnect.com)
# - Your deployed contract address (after deployment)

# 4. Compile contracts
npm run compile

# 5. Deploy to Base Sepolia
npm run deploy

# 6. Start dev server
npm run dev
```
